# SmartTradeAI

This is a self-intelligent, secure, and user-friendly web platform integrated with Zerodha Kite and TradingView APIs for automated and agentic algo trading.

## Setup

### Backend

1.  Navigate to the `SmartTradeAI/backend` directory.
2.  Install the required dependencies: `pip install -r requirements.txt`
3.  Run the backend server: `uvicorn app.main:app --host 0.0.0.0 --port 8000`

### Frontend

1.  Navigate to the `SmartTradeAI/frontend` directory.
2.  Install the required dependencies: `npm install`
3.  Run the frontend development server: `npm run dev`

## Deployment

### Frontend

The frontend is configured to be deployed on GitHub Pages.

### Backend

The backend can be deployed on any platform that supports Docker, such as Render, Fly.io, or Heroku.
